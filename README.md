# Console Java Chess

This project was developed as part of my studies in Object-Oriented Programming with Java.
It implements a complete chess game playable in the console, focusing on clean architecture, object modeling, and game rule implementation.

## Project Goals

- Practice advanced Object-Oriented Programming concepts
- Model a real-world system (chess rules) using classes and inheritance
- Implement game logic and rule validation
- Work with encapsulation and polymorphism
- Practice matrix manipulation and board representation
- Improve clean code and project organization

## Technologies Used

- Java 25
- Object-Oriented Programming (OOP)
- Git & GitHub

## Features

- Complete chessboard rendered in the console
- Movement validation for all chess pieces
- Capture system
- Turn-based gameplay
- Check detection
- Checkmate detection
- Move history tracking

## How to Run

1. Clone the repository
2. Compile the project
3. Run the application

## Concepts Practiced

- Object-Oriented Design
- Inheritance and Polymorphism
- Encapsulation
- Board game modeling
- Matrix manipulation
- Exception handling
- Game rule validation
